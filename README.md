⚡ BRUTAL.STORE

🎸 Unapologetic Fashion. 🚫 No soft edges. ⬛ No subtle gradients. 🔥 Just raw style.

👋 Welcome to BRUTAL.STORE, a fully responsive, front-end 🛒 e-commerce web application built entirely with Vanilla HTML 🦴, CSS 🎨, and JavaScript ⚙️.

🏗️ Designed with a bold Neo-Brutalist UI architecture, this project features stark contrasts ☯️, harsh borders ⏹️, high-impact typography 🔠, and zero compromises 🛑.

It operates without a backend, utilizing the browser's localStorage 💾 to simulate a complete database for user authentication 🔐, carts 🛍️, and order history 📦.
